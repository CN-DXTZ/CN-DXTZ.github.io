前言

若不会 Github 图床配置，参见上一篇《Github 图床 + jsDelivr CDN 图床加速》
本文选取插件 PicGo 实现一键上传图片至图床，并生成 MarkDown 语法的图片链接。
PicGo 有很多平台的版本，大同小异，由于本人使用 VScode 进行博客撰写，故选取的是 VScode 插件版。其他版本如 Windows 版亦可作参考。
